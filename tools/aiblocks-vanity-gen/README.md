# AiBlocks Vanity Address Generator

This folder contains `aiblocks-vanity-gen` a simple utility to generate vanity addresses that have some prefix.  This utility demonstrates the use of the
`GenerateRandomKey()` helper.

## Installing

```bash
$ go get -u github.com/aiblocks/go/tools/aiblocks-vanity-gen
```

## Running

```bash
$ aiblocks-vanity-gen PREFIX
```
